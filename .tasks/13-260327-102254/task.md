# Add a toKebabCase utility function

Create a simple utility function called toKebabCase that converts a camelCase or PascalCase string to kebab-case. Save to src/utils/to-kebab-case.ts with tests in src/utils/to-kebab-case.test.ts.

Examples:
- helloWorld → hello-world
- MyComponent → my-component
- alreadyKebab → already-kebab

---

Do not use bullet points, numbered lists, or any list formatting in replies. Write in plain prose sentences.

Why: The user prefers a more natural, flowing reading experience without list fragmentation.

How to apply: Every response should be continuous prose. If multiple items need to be mentioned, weave them into sentences with "and", "also", or "for example" rather than using "-" or "1." prefixes.
